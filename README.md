# 🚀 Abhay Kumar | Portfolio

Welcome to my personal portfolio! This is a high-performance, visually stunning website built with modern web technologies to showcase my projects, skills, and experiences.

## 🌐 Live Preview
[View Live Site](https://abhay-012-portfolio.vercel.app/)

---

## 🛠️ Tech Stack

- **Core:** React 19, TypeScript, Vite
- **Styling:** Tailwind CSS 4
- **Animations:** GSAP (GreenSock), Framer Motion
- **3D Elements:** Three.js, React Three Fiber, Drei
- **Utilities:** Lucide React, Lenis (Smooth Scroll), Clsx, Tailwind Merge

---

## ✨ Key Features

- **💎 Premium Design:** Modern, sleek interface with a focus on typography and spacing.
- **🌗 Dynamic Theming:** Smooth transitions between Light and Dark modes.
- **🎢 Advanced Animations:** Scroll-triggered animations using GSAP and Framer Motion for an interactive experience.
- **📦 3D Components:** Integrated Three.js elements for a modern, immersive feel.
- **📱 Fully Responsive:** Optimized for all devices, from mobile to ultra-wide monitors.
- **🚀 Ultra Fast:** Optimized with Vite and modern React features for lightning-fast performance.
- **🛰️ Contact Integration:** Functional contact form (Web3Forms) for direct communication.

## 🚀 Getting Started

Follow these steps to run the project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Abhaykumar99/New_Portfolio.git
   cd New_Portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
